Attach a render-texture to an attachment point.

Specifying a null texture will cause any existing attachment to be removed.

Parameters
attachment_pointA value from the AttachmentPoint enum.
textureThe texture to attach.
faceThe face of the texture to attach, as a value from the ITexture::Face enum.
z_offsetThe depth slice of the texture to write to, for 3D textures.
Returns
True on success, false on error.
