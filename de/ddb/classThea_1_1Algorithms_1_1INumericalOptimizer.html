Detailed Description

Interface for numerical optimizers, that minimize an objective function f:R^n–>R under given constraints.

To create an instance of a INumericalOptimizer, one typically loads the plugin for the relevant implementation and calls INumericalOptimizerFactory::createNumericalOptimizer().
