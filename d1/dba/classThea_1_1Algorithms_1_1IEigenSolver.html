Detailed Description

Eigenvalue solver interface.

The operator matrix must currently be real (and float64-precision), though complex eigenvalues/eigenvectors can be returned.

To create an instance of an IEigenSolver, one typically loads the plugin for the relevant implementation and calls IEigenSolverFactory::createEigenSolver().

Find the eigenvalues (and optionally eigenvectors) of a matrix, optionally requesting a particular number of eigenpairs.

A set of additional options may be specified to control the solution process.

Parameters
mThe operator matrix for the eigenproblem. Must be square.
compute_eigenvectorsIf false, only the eigenvalues will be computed, not the eigenvectors. getEigenvalue() will fail on subsequent calls.
num_requested_eigenpairsThe number of eigenpairs to look for, which need not be the number returned. A negative value implies that there is no upper bound on the number of eigenpairs.
optionsBackend-specific options. See the documentation of each derived class wrapping a particular backend.
Returns
The number of eigenpairs found, or a negative number on error.

Implemented in ArpackEigenSolver.
